# GEODES Text Editor

Éditeur de texte créé par le laboratoire <a href="http://geodes.iro.umontreal.ca/fr/">GEODES</a> de génie logiciel de l'université de Montréal.

## Installation
Aucune contrainte d'utilisation et d'installation.

## Langages
Cet éditeur de texte gère pour le moment seulement la coloration syntaxique du langage de programmation Java.

## Inspiration
Cet éditeur est inspiré de l'éditeur Pierre-Henry Soria : https://github.com/pH-7/Simple-Java-Text-Editor
